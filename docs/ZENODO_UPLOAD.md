# How to finish the Zenodo publish

Your reserved DOI: **10.5281/zenodo.21278092**

## Minimum (recommended): source archive only

Zenodo **Software** records do **not** require a PDF.

1. Open your Zenodo draft for DOI 10.5281/zenodo.21278092  
2. **Upload files** (drag and drop both if you want):
   - `dist/PyCHEMA-0.2.0-source.zip`  ← required
   - `dist/PyCHEMA_v0.2.0_Software_Note.pdf`  ← optional but nice
3. Or download the GitHub release zip:  
   https://github.com/Nikabanzai/PyCHEMA/archive/refs/tags/v0.2.0.zip  
   and upload that as the main file
4. Set **Resource type** = Software  
5. Title / description from `docs/ANNOUNCEMENTS.md`  
6. Related identifier: DOI `10.13140/RG.2.2.31672.55043` (Cites / Continues)  
7. License: Apache-2.0  
8. **Publish**

## Optional: attach PDF on ResearchGate too

Upload the same `PyCHEMA_v0.2.0_Software_Note.pdf` as full-text on a ResearchGate Software item.

## After publish

- Confirm https://doi.org/10.5281/zenodo.21278092 resolves  
- Badge in README (optional): Zenodo DOI badge
